# Customer Churn Prediction (Behavior-Based)

## Overview
This project predicts customer churn using behavioral data rather than
demographic information, simulating a real-world business use case.

## Dataset
Telco Customer Churn Dataset

## Approach
- Data cleaning and preprocessing
- Exploratory data analysis
- Feature engineering
- Machine learning modeling

## Results
The Random Forest model shows strong performance, with engagement and tenure
being the most important churn indicators.

## Future Improvements
- Survival analysis
- Time-based churn prediction
- Deployment with Streamlit
